# AI Resume Analyzer

An AI-powered ATS Resume Analyzer built using Flask, Python, HTML, and CSS.

## Features

- Upload Resume PDF
- Extract Resume Text
- ATS Score Calculation
- Skill Matching
- Missing Skill Detection
- Resume Suggestions

## Tech Stack

- Python
- Flask
- HTML
- CSS
- PDFPlumber

## Run Locally

```bash
pip install flask
pip install pdfplumber

python app.py
